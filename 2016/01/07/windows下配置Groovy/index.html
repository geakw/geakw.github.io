windows下配置Groovy

2016-01-07

windows下配置Groovy环境的教程网上很容易搜到，我参考的是这篇文章，安装过程一切顺利，然而在cmd中运行Groovy -v命令时返回ERROR: JAVA_HOME is set to an invalid directory，第一反应是我的JAVA_HOME变量配置的有问题，但cmd中运行java -version正常，应该不是这个变量的问题。

谷歌后发现这是Groovy的一个issue,这确实是有点坑，而且发现有这个问题还没有在最新的版本里面进行更改，so没办法，只能自己改了，废话不说，直接上步骤：

1、在Groovy根目录中进入bin文件夹，打开startGroovy bat文件

2、将下面一行


修改为 if exist “%JAVA_HOME%\bin\java.exe” goto valid_JAVA_HOME_DIR

然后运行Groovy -v，返回结果正常O(∩_∩)O~
